Step 1: Download

 

To run Neural Exploratorium, you must have the Java Runtime Environment version 1.2.2 or higher installed. If you do not have the JRE on your system, you can download it from here:

http://java.com/en/download/manual.jsp

 

If you already have JRE, download Neural Exploratorium from here:

http://www.nnod.com/NNT/neuron/Exploratorium.zip

 

Step 2: Run

 

Unzip Neural Exploratorium and double-click on the start1.bat file to initiate the application if you are using Microsoft Windows. If you are working under Unix, use /your_dirsctory/start.sh from Terminal Console

 

Step 3: Use Web Demo Page

 

You're now ready to launch Web Demo Page to try different demos now. Click on the following link to go to the Demo Page.

http://www.nnod.com/NNT/neuron/Exploratorium.htm

 

 

Шаг 1: Установка

 

Для работы Нейронной Экспериментальной Оболочки, на необходимо, чтобы на Вашем персональном компьютере была установлeна Java. Если Вы имеете навыки программирования, Вы можете установить ее самостоятельно, например из этого сервера:

http://java.com/en/download/manual.jsp

 

Если на Вашем компьютере уже имеется установленная виртуальная машина Java, Вам нужно переписать и распаковать Экспериментатор, который можно получить по этой ссылке:

http://nnod.com:8080/NNT/neuron/Exploratorium.zip

 

Шаг 2: Запуск

 

Нейронный Экспериментатор является Java-приложением. Если Ваш компьютер работает под управлением Windows, запустите start1.bat Если Вы используете UNIX, запустите start.sh используя соответствующую консоль оболочки.

 

Шаг 3: Работа с демонстрационной страницей

 

После того, как Экпериментатор запущен, Вы можете начать работу с ним, используя демонстрационную страницу:

http://nnod.com:8080/NNT/neuron/Exploratorium.htm
